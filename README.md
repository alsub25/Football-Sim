## Installation

1. Clone or download this repository
2. Open `index.html` in a web browser
3. For iPhone/iPad: Add to Home Screen for a native app experience

### Add to Home Screen (iPhone/iPad)
1. Open `index.html` in Safari
2. Tap the Share button (square with arrow)
3. Select "Add to Home Screen"
4. Tap "Add"

## Usage

### Playing a Quick Match
1. From the home screen, tap "Quick Match"
2. Select your home and away teams
3. Review team statistics and key players
4. Tap "Start Match"
5. Use the Play/Pause button to control the match
6. Watch events unfold in real-time
7. View final statistics and result

### Season Mode
1. Tap "Season Mode" from the home screen
2. Tap "Start New Season" to initialize the league
3. Use "Simulate Next Match" to play matches
4. Track team standings in the league table

## Game Mechanics

### Match Simulation Algorithm
The game engine simulates matches minute-by-minute using:
- Team attack, defense, and midfield ratings
- Random event generation based on team strengths
- Possession calculation from team attributes
- Shot quality determination for goal scoring
- Realistic event probabilities (goals, saves, fouls, corners)

### Scoring System
- Win: 3 points
- Draw: 1 point
- Loss: 0 points
